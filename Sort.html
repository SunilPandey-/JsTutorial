Best Link for Sort Example

By default, the sort() method sorts the values as strings in alphabetical and ascending order. This works well for strings ("Apple" comes before "Banana"). However, if numbers are sorted as strings, "25" is bigger than "100", because "2" is bigger than "1". Because of this, the sort() method will produce an incorrect result when sorting numbers. You can fix this by providing a "compare function"

function(a, b){return a-b}..
a and b are two of the values in the array, that you compare so Javascript can sort them. The function is called lots of times to determine where each element in the array is compared to all the others. The exact number of times the function is called depends on the number of elements in the array and their original order. You need to return 0 if the two elements are equal, a negative number if a should be before b and a positive number if b should be before a.
